## 주문 패턴 — 엑셀 파일 발주 (일별 파일)

윌리스요양병원은 **엑셀 파일(.xlsx)** 로 발주합니다. 검수일지 겸용 발주서 형식입니다.

### 엑셀 템플릿 구조

| 위치 | 내용 | 비고 |
|:----:|------|------|
| 파일명 | `월5-25.xlsx` (요일M-D) | 요일약자+월-일 |
| 시트명 | `월발주` (단일 고정) | 변경되지 않음 |
| R3(G7) | `2026-05-25 00:00:00` | **입고일** (필수 파싱) |
| R4 | 헤더: 품명/검수/규격/입고수량/단위/업체명/... | |
| R7~R42 | **1차 발주 품목** (35개) | 주발주 |
| R44~R51 | **2차 발주 품목** (5개) | 추가/별도 발주 |
| R74+ | 검수일지 | 발주 데이터 아님, 제외 |

### 컬럼 매핑
| 열 | 헤더명 | 매핑 | 비고 |
|:--:|--------|:----:|:----:|
| A(1) | 품 명 | **name** | |
| B(2) | 검수 | 무시 | |
| C(3) | 규격 | **일반 품목: 무시** | `10k`, `700g` 등 제품 표준규격. 단, `고무장갑`/`면장갑`은 색상/사이즈 정보이므로 품명에 포함 (아래 규칙 참고) |
| D(4) | 입고수량 | **qty** | |
| E(5) | 단위 | **unit** | `박`=박스, `K`=kg |
| F(6) | 업체명 | **입고일 지정 + 품명보조** | `화입고` 등 입고일 지정값이 아니면 고무장갑/면장갑의 제조사 또는 **다진마늘의 원산지 정보**를 품명에 결합 |
| G~ | 유통기한~ | 무시 | |

### ⚠️ 단위 정규화
- **`박` = `박스`** (box). 동일 의미로 처리
- 예: `맛김치 1 박` = `맛김치 1 박스`

### ⚠️ 업체명(F열) 특수값 — 입고일 지정
F열(업체명)에 일반 업체명 대신 아래 값이 들어오면 **해당 품목의 입고일이 다름**을 의미:

| 값 | 의미 | 처리 |
|:--:|:----|:----:|
| `화입고` | **화요일 입고** (월→화, D+2) | 해당 품목의 입고일 = 파일날짜+1 |
| `수입고` | 수요일 입고 | |
| `완제` | 완제품 (가공완료) | 일반 발주와 동일 처리 |
| `국산` | 원산지 정보 | 무시 |
| `화입고` | 공란 외 일반 텍스트 | 무시 |

예: 5/25(월) 파일에서 `절단꽃게s`에 `화입고` → 5/26(화) 입고

### ⚠️ 고무장갑·면장갑 품명 규칙 (윌리스 전용)

고무장갑과 면장갑은 A열(품명)만으로 구분이 불가능하므로, **C열(규격)과 F열(업체명)의 값을 A열 품명에 결합**하여 name을 구성하세요.

**규칙:** `A열값(C열값 F열값)` 형식으로 name 생성

| A열 (품명) | C열 (규격=색상/사이즈) | F열 (업체명=제조사) | 추출할 name |
|:----------:|:----------------------:|:-------------------:|:-----------:|
| `고무장갑` | `노랑/S` | `크린랲` | `고무장갑(노랑/S크린랲)` |
| `고무장갑` | `분홍/S` | `태화` | `고무장갑(분홍/S태화)` |
| `고무장갑` | `빨강/M` | `태화` | `고무장갑(빨강/M태화)` |
| `고무장갑` | `노랑/M` | `후디스` | `고무장갑(노랑/M후디스)` |
| `면장갑` | `초록2줄` | (빔) | `면장갑(초록2줄)` |

**참고:** 위 패턴의 품명이 DB(품명1)에 이미 등록되어 있어 정확히 일치하면 자동 매칭됩니다.

### ⚠️ 다진마늘 품명 규칙 (원산지 구분)

다진마늘은 A열(품명)만으로 `국산`/`중국산` 구분이 불가능하므로, **F열(업체명)의 원산지 정보를 A열 품명에 결합**하여 name을 구성하세요.

| A열 (품명) | F열 (업체명=원산지) | 추출할 name |
|:----------:|:-------------------:|:-----------:|
| `다진마늘` | `중국산` | `다진마늘(중국산)` |
| `다진마늘` | `국산` | `다진마늘(국산)` |

**참고:** DB에 `다진마늘(중국산)` / `다진마늘(국산)`으로 이미 등록되어 있어 정확히 일치하면 자동 매칭됩니다.

### 발주 시간대
- **엑셀 발주:** 오전 8시~10시 사이 (매일)

```
[박운재] 파일: 수4-23.xlsx
[박운재] 파일: 목4-24.xlsx
[박운재] 파일: 금4-25.xlsx
```

- **파일명 형식:** `요일약자M-D.xlsx` (초기) → `M-D 요일.xlsx` (후기)
  - 요일 약자: 월, 화, 수, 목, 금, 토
  - 예: `수4-23` = 수요일 4월 23일, `1-5 월.xlsx` = 1월 5일 월요일
- **매일 파일 전송:** 입고일마다 별도 파일 (태종대요양병원은 여러 날짜를 한 파일에)
- **도 인호 응답:** `특이품목 없습니다~` 또는 `나머지 특이품목 없습니다~` (정형화된 확인)

### 2️⃣ 추가 주문 — 텍스트 메시지

```
설탕 3봉같이 입고 부탁드립니다
맛김치 10킬로 추가입니다~~총20킬로요
포기김치 20kg 추가로 입고부탁드립니다
물엿 4통이랑 오이 1kg도 추가부탁드려요
카스테라 6박스 주문해서 같이 입고 부탁드립니다
```

- **키워드:** `입고부탁드립니다~`, `추가입니다`, `같이 입고`
- 단위: 봉, 박스, kg(킬로), 개, 통, 판, 망, 팩
- 여러 품목 동시 추가 가능: `물엿 4통이랑 오이 1kg도`

### 3️⃣ 배송일 지정 추가 주문

```
우유 85개, 두유 10개 4/26(토)에 입고해주세요!
닭가슴살 2kg 4/28(월) 입고부탁드립니다
카스테라(80개입) 2박스 2/12(목)에 입고부탁드립니다
```

- 추가 주문 시 **날짜 + 요일**을 명시하는 경우가 많음
- 형식: `M/D(요일)에 입고해주세요`

### 4️⃣ 주문 취소

```
마요네즈 취소입니다
두부1판 취소입니다
시금치 0.5kg 취소 → 5.5kg
```

- **형식:** `[품목] 취소입니다` / `[품목] 취소`
- 취소하면서 수량 정정도 가능: `시금치 0.5kg 취소 → 5.5kg`

### 5️⃣ 선발주 확인

```
선발주 품목 있는지 알 수 있을까요~~?
혹시 파슬리가루 선발주 품목일까요?
```

- D-1, D-2 발주 품목 사전 확인
- 도 인호 응답: `D-1발주입니다~~`, `D-2발주입니다~`

### 6️⃣ 온라인 쇼핑몰 링크 주문

```

```

- 특정 상품을 네이버 스마트스토어/쿠팡 링크로 주문
- 호스트가 링크 보고 직접 구매 후 배송

### 7️⃣ 품목별 배송일 지정

```
우유 85개, 두유 10개 4/26(토)에 입고해주세요!
```
