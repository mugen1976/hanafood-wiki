[가영식당]이 품목과 수량을 **공백으로 구분**하여 전송:

```

```

- **형식:** `품목명수량단위` 를 공백으로 구분
- 여러 줄로 나누어 보내기도 함
- 품목과 수량이 붙어있음 (예: `콩나물1만원`, `간마늘1봉`)

### 4️⃣ 주문 수량 단위
가영식당은 다양한 단위 사용:
- `키로` (kg): 산취나물2키로, 꽈리고추2키로
- `봉지`/`봉`: 줄줄이비엔나3봉지, 카레2봉지, 다시멸치2박스
- `박스`: 수입김치1박스, 무우1박스
- `통`: 고추장1통, 식용유1통, 트리오락스1통
- `단`: 단배추10단, 대파2단, 마늘쫑3단
- `판`: 두부1판, 도토리묵1판
- `만원` (금액): 콩나물1만원, 오이2만원 (만원어치)
- `근`: 고춧가루5근
- `g` / `kg`: 간생강500g

### 5️⃣ 두부 반판/판반 변환 규칙
가영식당은 "두부반판" (두부 반 판 = 0.5판) 표현을 사용합니다.
- **`두부반판`** → `두부1팩` 으로 자동 변환
- **`두부1판반`** → `두부1판` + `두부1팩` 으로 자동 분할
- (`판`=block, `팩`=pack, 1판 = 2팩)
