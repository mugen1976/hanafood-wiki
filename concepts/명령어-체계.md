# 명령어 체계

발주방에서 사용되는 봇 명령어 정리.

## 발송 명령어

| 명령어 | 설명 |
|--------|------|
| `!발송확인` | 미리보기에 등록된 모든 매입처로 발송 실행 |
| `!발송확인 N` | 지정한 번호의 매입처만 발송 (예: `!발송확인 1`) |
| `!발송확인 N,M` | 여러 매입처 선택 발송 (예: `!발송확인 1,3`) |
| `!발송확인 N M` | 공백으로도 구분 가능 (예: `!발송확인 1 3`) |
| `!재발송` | 미확인/실패한 항목만 다시 발송 |
| `!발송취소` | 현재 미리보기/발송 취소 (유효시간 300초) |

## 확인 및 응답

봇의 응답 메시지:
- `!발송확인 1` → 발송 실행 후 `[발송 결과]` 리포트 출력
- 각 매입처별 발송 건수와 상태를 상세히 표시
