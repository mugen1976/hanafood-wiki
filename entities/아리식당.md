## 주문 패턴

### 1️⃣ 주문 방식 — "붙여쓰기" 스타일
[아리식당]이 품목과 수량을 **띄어쓰기 없이 붙여서** 한 줄로 전송:

```
램33ㅡ1개두부1무1청양고추1국거리3마요네즈2부지갱이6간마늘2양상추1적채2콩나물5고비1양파1
```

- **형식:** `품목명수량`
- 수량 단위: 개, 박스, 1(기본), k(kg), 망, 봉지, 판, 대
- 여러 품목을 공백 없이 연속해서 나열
- **드물게 띄어쓰기 사용:** `양파 1망`, `애호박 10개`, `시금치 2박스`

**변화:** 2026-05-18부터 띄어쓰기 사용 증가:
```
김치 2 북어채 1 동초 7 도라지 4 쇼리 1 지고추 5
```

### 2️⃣ 추가 주문 (분할 발송)
기존 주문과 별도 메시지로 추가 품목 전송:

```
야채고로게1박스
두부1
설탕1추가요
강정2추가요
돈가스1박스추가요
```

- **키워드:** `추가요`, `추가해주세요`, `추가줌해주세요`
- 단순 품목명만 보내면 추가로 간주됨
- 별도 시간에 추가 품목 전송 가능 (예: 6시 32분 주문 → 6시 52분 추가)

### 3️⃣ 주문 취소
```
돈가스취소해주세요
```
- 취소하려는 품목명 + `취소해주세요` 형식

### 4️⃣ 정육(고기) 주문 특징
정육(돼지고기)은 별도로 주문하며 용도별 구분:
```
정육20
국거리8
찌게용8
정육20국거리6찌게용6고기는월요일주세요
```
- 용도: 국거리(국/찌개용), 찌게용, 불고기용, 제육용
- 배송일 지정: `고기는월요일주세요`, `내일보내주세요`
- `정육20` = 20kg

### 6️⃣ 부재/휴무 안내
```
내일부터 8월3일까지. 휴가입니다
일이 없어서 금토일 쉽니다
```

### 7️⃣ 기타 상호작용
- **발주 요청:** [Ray] 또는 [도 인호]가 `발주 주세요`, `주문 주세요`로 주문 독려
- **정정:** `물미역이아니고미역줄기요` (잘못된 품목 정정)
- **품질 요청:** `양상추좋은것으로보내주세요`

### 8️⃣ 암묵적 단위 규칙 (단위 생략 시)
단위를 생략한 경우 수량에 따라 추정 단위가 달라짐:

- **콩나물:** 수량=1 → `통`, 수량≥2 → `kg`
  - 예: `콩나물1` = 콩나물 1통, `콩나물6` = 콩나물 6kg
