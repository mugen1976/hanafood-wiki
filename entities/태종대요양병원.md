## 주문 패턴 — 엑셀 파일 발주

태종대요양병원은 **엑셀 파일(.xlsx)** 로 주문합니다.
가영/아리식당과 완전히 다른 방식입니다.

### 엑셀 템플릿 구조

- **시트명 = 요일** (예: `수요일`, `금요일`) — 날짜가 직접 쓰이지 않음
- **A3 셀**에 실제 날짜 기록:
  ```
  2026-5-22(금)발주 / 5-25(월)입고
  ```
  → 앞쪽 = 발주일, 뒤쪽 = 입고일 (입고일을 발주 대상일로 사용)
- **발주 시점 = 입고일 하루 전** (D-1). 예: 5/22(금)발주 → 5/25(월)입고

### 1️⃣ 정기 발주 — 엑셀 파일

```
[정원주] 파일: 4월 21 23 25  .xlsx
[정원주] 4월 23일 수요일 입고분 발주입니다
```

- **주기:** 월/수/금 정기 입고 (일부 토요일 입고 있음)
- **파일명:** `M월 D D D .xlsx` 형식 (입고일 나열)
- **날짜 기준:** 엑셀 시트에 입고일별로 정리
- **입고 요일:** 기본 월(月)·수(水)·금(金), 가끔 토(土)

### 2️⃣ 추가 주문 — 텍스트 메시지

엑셀 발주 후 별도 메시지로 추가:

```
새우살 2봉 추가해주세요
무 1박스 추가해주세요
감자 1박스 추가발주합니다
오이 1박스 추가될까요?
```

- **형식:** `[품목] [수량][단위] 추가해주세요 / 추가발주합니다`
- **키워드:** `추가해주세요`, `추가발주합니다`, `추가될까요`
- 단위: 봉, 박스, kg(키로), 팩, 판, 망, 통, 말

### 3️⃣ 주문 취소

```
무 1박스 취소해주세요
부추 1단 취소해주세요
```

- **형식:** `[품목] [수량][단위] 취소해주세요`
- 간혹 엑셀 파일 자체 수정도 있음

### 4️⃣ 선발주 (미리 주문)

```
금요일 입고분 선발주 입니다
바나나 8손짜리 1박스, 크고좋은 바나나 1손 입고 부탁드립니다
```

- 입고일보다 먼저 주문 (예: 화요일에 금요일 입고분 선발주)

### 5️⃣ 가격 문의 (매우 빈번)

```
사장님~콩나물 가격이 얼마인가요?
사과 1K 얼마인가요?
고등어, 동태, 가자미 가격부탁드립니다
```

- 도 인호가 시세를 알려주면 그에 따라 발주 조정
- 응답 패턴: `금일 kg X,XXX원입니다~`, `D-1발주입니다~`

### 6️⃣ 수량/품목 정정

```
동초 취소하고 추가발주합니다. 오이 1박스 부추 0.5k
감자 1박스에서 1박스로 변경하겠습니다
참치캔 2 발주로 변경하겠습니다
```

### 7️⃣ 품질 클레임 (사진 동봉)

```
사진 — 호박이 이런게 들어와요 / 무 상태 안좋음 / 방울튀김 소비기한 지남
```

- 사진으로 불량 상품 증거 첨부
- 도 인호가 보상/교환 처리

### 8️⃣ 기타 상호작용
- **품질 클레임 (사진 동봉):** 불량 상품 증거 사진 → 도 인호가 보상/교환 처리
