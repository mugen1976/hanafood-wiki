## 기존 주문 패턴 (텍스트 메시지)

과거에는 텍스트 메시지로 발주했으나, 현재는 엑셀 파일 발주가 주력입니다. 간혹 추가 주문/취소 메시지가 옵니다.

### 복합 메시지 형식 (추가분 + 취소분)

당당한방병원은 **추가분/취소분 섹션이 명시적으로 구분된 멀티라인 메시지**를 사용합니다:

```
M/D(요일)
하나푸드
추가분
품목1수량, 품목2수량, ...

취소분
품목1수량, 품목2수량, ...
```

**예시:**
```
5/28(목)
하나푸드
추가분
흙감자5kg, 양파1망

취소분
고구마1박스, 노피망0.5kg, 홍피망0.5kg, 왕란5판
```

**구조:**
| 라인 | 내용 |
|------|------|
| 1 | `M/D(요일)` — 발주일 |
| 2 | `하나푸드` — 고정 접두사 |
| 3 | `추가분` — 추가 섹션 헤더 |
| 4~ | 추가할 품목들 (콤마 구분) |
| 빈 줄 | 구분 |
| 다음 | `취소분` — 취소 섹션 헤더 |
| 이후 | 취소할 품목들 (콤마 구분) |

**다중 날짜 연속 메시지도 가능:**
```
5/22 (목)
하나푸드
추가분
고구마1박스

취소분
흙감자1박스

5/23(금)
추가분
흙감자1박스, 왕란5판
```

**주의사항:**
- `추가분`/`취소분` 키워드는 각각 **독립된 줄**에 있어야 함 (휴리스틱 파싱에서 섹션 구분자로 사용)
- 품목은 콤마(`,`)로 구분, 각 품목은 `이름+수량+단위` 형식
- 날짜는 `M/D(요일)` 괄호 형식이 기본
