### ⚠️ 중요 파싱 규칙
1. **입고일 위치**: A3 셀 (`2026년 5월 25일 (월요일)`)
2. **D+3 패턴**: 금요일(5/22) 발송 → 월요일(5/25) 검수 (주말 포함 3일)
3. **시트명 `M.D`**: `5.25` = 5월 25일 (현재 연도)
4. **`kfg` → `kg`** 단위 오타 정규화
5. **숫자 `,` = 소수점** — 병원 측에서 `.` 대신 `,`를 잘못 사용. `0,6` = 0.6, `2,6` = 2.6, `1,6` = 1.6
6. **중간 헤더 skip**: `유통온도` 또는 안내문구가 데이터 중간에 있을 수 있음
7. **번호만 있는 행 skip**: 품목명 없이 번호만 있는 행은 빈 행
8. **과거 시트 skip**: `금요일(12.15)` (2023년), `금요일(1.8) (2)` (2024년) 등 과거 시트는 발주 대상 아님
9. **C열은 `규격` 정보** — 수량이나 단위로 착각하지 말 것
10. **비고(L열) 특수 규칙**:
    - **`수요일납품`** → 해당 품목 delivery_date를 기본 검수일+2일(수요일)로 오버라이드
    - **`N+N'kg'` 패턴** (예: `1+1,6kg`) → 분할 발주. 콩나물 2.6kg → 콩나물 1kg + 콩나물 1.6kg
